Brentford Analytics Automation
Automated ETL Pipeline & Performance Scraper
Project Overview
This project automates the extraction and normalization of high-stakes football performance data. By replacing manual data entry with a Python-based ETL (Extract, Transform, Load) pipeline, this tool ensures 100% data integrity and significantly reduces reporting overhead for stakeholders.

Key Features
Automated Web Scraping: Uses BeautifulSoup to pull real-time league standings.

Data Normalization: Leverages Pandas to clean "messy" HTML data and handle missing values.

Stakeholder Delivery: Automatically exports cleaned data into structured CSV format.

Impact
This workflow mirrors the processes I managed during my 5+ years at Brentford FC, where I utilized similar custom frameworks to achieve 98% predictive accuracy in talent performance assessments.
